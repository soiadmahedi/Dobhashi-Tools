# Dobhashi Tools

Welcome to the official release repository for **Dobhashi Tools**, a premium application designed to help you learn the Sylheti Nagri script, translate text, and explore a comprehensive dictionary.

**Developed by:** Soiad Mahedi  
**Website:** [soiadmahedi.top](https://soiadmahedi.top/dobhashitools/)

> ⚠️ **Note:** This is a closed-source project. This repository is solely used for distributing the latest releases and tracking user issues/bugs.

## 📥 Download the Latest Version

You can download the latest compiled versions from the [Releases](../../releases) page:
* 📱 **Android:** Download the `.apk` or get it on Google Play.
* 💻 **Windows:** Download the `DobhashiTools_Setup.exe` for PC.
